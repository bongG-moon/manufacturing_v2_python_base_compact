# Manufacturing Agent Harness

이 폴더는 제조 분석 에이전트의 **기준 하네스 구조**입니다.

이 하네스가 다루는 범위:
- 제조 도메인 지식 구조
- 데이터셋 카탈로그
- 질문 유형과 분석 패턴
- 실행 계획
- 평가 기준
- 실제 구현 리포지터리와의 동기화 방법

참고 소스:
- `manufacturing_v2_python_base`
- `manufacturing_v2_python_base_compact`

이 하네스의 목표:
- 모델이 바뀌어도 일관된 구조를 유지
- 초보자도 운영 가능한 문서 중심 체계 제공
- 실제 리포지터리에서 재사용 가능한 기준점 제공

# Evaluation

## 목적
제조 에이전트의 출력 품질을 계속 검증하기 위한 기준입니다.

## 평가 축
- 도메인 용어 추출 정확도
- 데이터셋 선택 정확도
- 후속 분석 코드 생성 정확도
- 파생 지표 계산 정확도
- 답변과 테이블의 grounding 일치 여부

## 최소 평가 세트
1. 단일 조회 질문
2. 다중 데이터셋 조회 질문
3. 후속 groupby 질문
4. top-N 질문
5. 달성율 질문
6. 불량률 질문
7. 없는 컬럼 요청 질문

## 조직 맞춤 평가 포인트
- 답을 안 하는 것이 나은지, 낮은 확률이라도 답하는 것이 나은지
- 보수적 응답 vs 탐색적 응답
- 도메인 지표의 우선순위

# Manufacturing Harness Bootstrap Plan

## 목표
제조 에이전트용 하네스 기준 구조를 만들고, 실제 구현 리포지터리에서 재사용 가능한 상태로 만든다.

## 작업 항목
1. 서비스 폴더 구조 생성
2. AGENTS.md 및 docs 기본 세트 작성
3. 동기화 스크립트 작성
4. 검증 스크립트 작성
5. 실제 구현 리포지터리 연결 규칙 문서화

## 완료 기준
- `services/manufacturing-agent/` 구조가 존재
- 핵심 문서가 모두 존재
- pull / push 스크립트가 동작
- validate 스크립트가 동작
